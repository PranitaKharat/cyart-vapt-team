# VAPT Practical Tasks

This repository contains practical work related to Vulnerability Assessment and Penetration Testing (VAPT).

## Tasks Completed

1. Advanced Exploitation
2. Web Application Testing
3. Evidence Collection
4. Security Reporting
5. Capstone VAPT Project

## Tools Used

- Kali Linux
- Nmap
- Metasploit
- Burp Suite
- SQLMap
- Wireshark

## Screenshots

Screenshots of scanning, exploitation, and testing activities are included in their respective folders.

## Outcome

The project demonstrates the process of identifying vulnerabilities, exploiting them, collecting evidence, and preparing a professional security report.
