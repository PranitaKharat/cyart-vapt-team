# Post Exploitation and Evidence Collection

## Objective
The objective of this task is to collect evidence after successful exploitation of a vulnerable system.

## Tools Used
- Metasploit
- Netcat
- Wireshark
- Kali Linux

## Activities Performed

1. Accessed the vulnerable machine using an exploit.
2. Established a shell session with the target system.
3. Collected system information from the compromised machine.
4. Captured network traffic using Wireshark.
