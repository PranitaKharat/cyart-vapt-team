# Capstone Project: Full VAPT Cycle

## Objective

The objective of this project is to simulate a full penetration testing cycle including reconnaissance, vulnerability scanning, exploitation, and reporting.

## Methodology

The penetration testing process followed standard security assessment phases:

1. Reconnaissance
2. Scanning
3. Exploitation
4. Post Exploitation
5. Reporting

## Tools Used

- Kali Linux
- Nmap
- Metasploit
- Burp Suite
- SQLMap

## Activities Performed

1. Identified open ports and services using Nmap.
2. Analyzed vulnerabilities in the target system.
3. Exploited vulnerable services using Metasploit.
4. Tested web application vulnerabilities using Burp Suite and SQLMap.
5. Collected evidence from compromised system.
